# Local Agent KV Cache Benchmark

## Confirmed Python Environment

The working code environment has been tested in this repo.

- Primary command: `.venv/bin/python`
- Python: `3.11.14`
- Backing conda environment: `tunnel-agent`
- Conda path: `/opt/miniconda3/envs/tunnel-agent`
- Tested plotting packages: `pandas 3.0.3`, `matplotlib 3.10.9`

The `.venv` was created from the `tunnel-agent` conda environment. Prefer using
`.venv/bin/python` from the repository root so Matplotlib and project imports use
the same environment.

## Regenerate Report Figures

The technical report figures are generated by:

```bash
.venv/bin/python scripts/plot_report_figures.py
```

This writes PNG and PDF outputs to:

```text
results/figures/report/
```

The current plotting convention is:

- Figure titles are not embedded inside the plots.
- The paper/report caption provides the figure title.
- Axes labels, tick labels, legends, and key value annotations are enlarged for
  readability at `\linewidth` in the LaTeX report.
- Plot styling is high-contrast, clean, and publication-oriented.

To refresh the LaTeX report after regenerating figures:

```bash
cp results/figures/report/report_*.pdf reports/kv_cache_experiment_report/figures/
cd reports/kv_cache_experiment_report
latexmk main.tex
```
